# Security Policy

## Supported Scope

Security reports are accepted for the main branch (`main`) and current release state.

## Reporting a Vulnerability

Please report security vulnerabilities privately to:
- `dolph.hincapie26@gmail.com`

Include:
- affected area/file,
- reproduction steps,
- impact assessment,
- possible mitigation (if available).

## Response Goals

Target response times (best effort):
- Initial acknowledgement: 72 hours
- Triage decision: 7 days
- Fix timeline: based on severity and complexity

## Disclosure Policy

- Do not open public issue for unpatched vulnerabilities.
- Wait for maintainer response and coordinated disclosure.
