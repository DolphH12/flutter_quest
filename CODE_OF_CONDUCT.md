# Code of Conduct

## Our Commitment

We are committed to making participation in Flutter Quest respectful and inclusive for everyone.

## Expected Behavior

- Be respectful and constructive.
- Focus on technical arguments, not people.
- Accept feedback and collaborate in good faith.
- Use clear, professional communication.

## Unacceptable Behavior

- Harassment, insults, or discriminatory language.
- Personal attacks or intimidation.
- Repeated disruptive behavior in issues/PRs.
- Publishing private information without consent.

## Enforcement

Project maintainers may remove, edit, or reject contributions and interactions that violate this code.

## Scope

This code applies to:
- GitHub issues,
- pull requests,
- discussions,
- project-related communication spaces.
